# TriByte Symposium 2026

🎯 National Level Technical Symposium at C. Abdul Hakeem College of Engineering and Technology

## 📋 About
TriByte is a technical symposium featuring 9+ events across Technical and Non-Technical categories. 
This platform allows students to register for events, make UPI payments, and track their registrations.

## ✨ Features
- User Authentication (Register/Login)
- Event Listing with Real-time Registration Counts
- Team Registration with Multiple Participants
- UPI Payment Integration with QR Code
- Dashboard with Registration History
- Ticket Download as HTML
- Google Sheets Integration for Backup

## 🛠️ Tech Stack
**Frontend:**
- React.js
- React Router
- Axios
- CSS3

**Backend:**
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Google Sheets API
